## Getting Started

Before starting, make sure you have Android Studio installed.  

### 1. Initial Step:

```bash
cd \.locationTarget
git clone https://github.com/Opjasson/learingManagementSystem-Application.git
```

```file
1. open Android Studio software
2. open folder learningManagementSystem-Application in android studio
2. wait until the graddle installation is complete
```

```backend
-- Database firestore
1. create firestore database in https://console.firebase.google.com/
2. adjust the collection, document and database fields to the contents of the Modal file
3. connect firestore with android studio
4. add the cloud firestore SDK to your app

-- Authentikasi Firebase
1. open the folder where the application you have created is located in https://console.firebase.google.com/
2. open build menu then open Authentikasi menu
3. turn on authentikasi firebase
4. adjust according to your project needs
5. open android studio
6. connect authentikasi with custom email and password
7. add the cloud firebase authentikasi SDK to your app
```
```finish
Please try to run the application in Android Studio.
```
